## তোমার কাজের তালিকা

### 0. পড়ো
- `CONTINUE_PROMPT.txt` — সম্পূর্ণ Plan 16, ডায়াগনোসিস, ফাইল ম্যাপ
- মূল ফাইল: `src/lib/textReflow.ts`, `src/components/studio/FabricLines.tsx`, `src/state/overridesStore.ts`, `src/state/linkingStore.ts`, `src/components/studio/PropertiesPanel.tsx`

### 1. ইমপ্লিমেন্ট করো (Plan 16 ধাপ অনুযায়ী)
1. `src/lib/reflowScope.ts` — `effectiveReflowScope(scope, layer, linking)`
2. `src/lib/textReflow.ts` — `reflowLayerText()` + আরবি split (`splitArabicWords`)
3. `overridesStore.patchScoped` — typography (`fontPx`, `leading`, `tracking`, `hScale`) পর reflow ট্রিগার
4. `FabricLines.tsx` — `getReflowBase` / `checkOverflow` / Enter — একই scope + linking লজিক
5. `LinkingPanel` — Area Text হিন্ট (লিংক ON/OFF ব্যাখ্যা)
6. `scripts/verify-reflow.mjs` — Playwright E2E (font বড় → পরের সারিতে টেক্সট)

### 2. বাগ চেক
- লিংক OFF → overflow অন্য সারিতে যাবে না
- লিংক ON + scope=surah → typography reflow সূরা জুড়ে
- ক্রস-পেজ → `CrossPageReflowDialog` (বিদ্যমান) রিইউজ
- React hooks — early return এর আগে

### 3. ভেরিফাই
```powershell
npx tsc --noEmit
npm run build
node scripts/verify-editor.mjs
node scripts/verify-reflow.mjs
```

### 4. Git
```powershell
git add -A
git commit -m "feat(Plan16): linked area-text auto reflow on typography and scope"
git push origin main
```

### 5. হ্যান্ডঅফ আপডেট
- `CONTINUE_PROMPT.txt`-এ Plan 16 ✅ চিহ্নিত করো
